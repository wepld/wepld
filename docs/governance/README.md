# WePLD Governance

Internal governance package for WePLD's proprietary intellectual-property
posture (adopted 2026-07-19). These are internally adopted governance
records, not legal advice; documents intended to bind an external party
require qualified legal review in the relevant jurisdiction(s) before
use.

Root records: `LICENSE` (all-rights-reserved proprietary notice),
`COPYRIGHT` (ownership record), `CONTRIBUTING.md` (contributions-closed
posture) — all at the repository root.

## Policies

| Document | Subject |
| --- | --- |
| `INTELLECTUAL_PROPERTY.md` | ownership model, kinds of rights, deferred legal matters |
| `AI_ASSISTED_DEVELOPMENT.md` | human ownership, provenance, provider restrictions |
| `THIRD_PARTY_DEPENDENCIES.md` | inbound license categories and approvals |
| `DEPENDENCY_REGISTER.md` | recorded dependencies and review status |
| `CONFIDENTIALITY.md` | classifications, controls, recorded risks |
| `REPOSITORY_ACCESS_CONTROL.md` | access, least privilege, protected refs |
| `INVENTION_LOG.md` | invention records and pre-publication review |
