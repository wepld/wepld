# Invention Log

**Status:** internally adopted (GDR-008, 2026-07-19). Lightweight
internal record. **This log is not a patentability assessment**, and
nothing in it constitutes legal advice or a filing decision. Its purpose
is to preserve options: public disclosure before filing can create
patent bars in many jurisdictions, so potentially novel work is logged
and reviewed **before** any public disclosure.

## Pre-publication invention-review gate

Before any publication, talk, demo, paper, or other public disclosure of
WePLD technical material, the founder reviews this log for affected
entries. An entry with `public_disclosure_status: NotDisclosed` and
potential novelty requires an explicit founder decision (and, where
warranted, qualified patent counsel review) before the disclosure
proceeds. See GDR-008 for the full gate.

## Entry template

```text
- id: INV-NNN
  date: YYYY-MM-DD
  inventor_or_contributors: <names>
  problem: <what problem the work addresses>
  proposed_technical_solution: <how it is solved, briefly>
  potentially_novel_elements: <what might be new, without any
    patentability claim>
  public_disclosure_status: NotDisclosed | PartiallyDisclosed | Disclosed
    (with date and venue if disclosed)
  third_party_materials_involved: <dependencies, prior art, external
    code or data, or "none known">
  founder_review: Pending | Reviewed (date, outcome)
  legal_or_patent_review_status: NotRequested | Requested | Completed
    (date, counsel, outcome)
```
